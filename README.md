# 五子棋
> 在线五子棋

### 特性
- 创建房间、加入房间、离开房间
- 游戏重开(仅房主)
- 自动转移房主(对方退出房间或掉线)
- 随机先手
- 多人观战
- 悔棋(需对手同意)

### 开始
```
go run cmd/main.go
```
